# 🐳 Game: Where's Popo?

### 1. 서비스 소개(Intro)

제한 시간 내에 상단에 위치한 이미지와 같은 이미지를 찾아내면 이기는 게임입니다.

</br>

### 2. 기능 소개(Features)

- VanillaJS 로 구현한 미니 게임
- 시간대를 감지하여 라이트 모드 / 다크 모드로 전환 및 유지
- mediaQuery를 통해 반응형 UI 구현

</br>

### 3. 사용 스택(Skills)

Vanilla JS, HTML, CSS

</br>

### 4. 배포 링크(Preview)

https://popo-game.netlify.app

</br>
